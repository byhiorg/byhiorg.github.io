将本站的评论系统gitment改为disqus了

发表于 2018-10-01 更新于 2018-10-12 分类于 综合 阅读次数：

本博客原来的评论系统为 Gitment

Gitment 是一款基于 GitHub Issues 的评论系统,可以使用 GitHub 账号进行评论。

我主题 Next 自带的服务接口已经停止维护了，而我又不想购买服务器自己搭一个CORS header server。所以改用disqus了。

对于想要自己创建评论服务的可以看这里：https://github.com/imsun/gh-oauth-server 作者已经把这个开源了，很简单的 nodejs 代码。

因为disqus评论系统为国外服务。如果长时间无法加载，请针对 disq.us | disquscdn.com | disqus.com 启用代理。

于更换日(2018-10-01)简单记录下。
